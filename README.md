# Petrol Bunk Management System

A Java Full Stack web application developed using Spring Boot, JSP, and MySQL to manage petrol bunk operations efficiently.

## Features

* Admin Management Module
* Branch Management Module
* Fuel Inventory Management
* Billing & Payment Management
* User Authentication System
* CRUD Operations
* Responsive User Interface

## Modules

### Admin Module

* Manage branches
* Monitor fuel inventory
* Manage users and reports

### Branch Manager Module

* Update fuel stock
* Manage transactions
* Monitor branch operations

### User Module

* View fuel details
* Access transaction information
* Manage profile details
